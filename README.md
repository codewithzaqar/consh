# Consh
A simple Python-based CLI insoired by xonsh

## Installation
```bash
pip install .
```

## Usage 
Run the CLI
```bash
consh
```
Available commands:
- `hello [name]`: Prints a greeting.
- `env [key]`: Shows environment variables.
- `exit`: Quits the CLI.
- System commands (e.g., `ls`, `pwd`) are supported.

## Development
- Python 3.6+
- No external dependencies
